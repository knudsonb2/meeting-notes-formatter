TEMPLATE FOLDER
===============

Place your meeting notes template files (.docx) here.

Files placed in this folder will automatically appear as selectable 
templates in the application.

The filename (without extension) will be used as the button label.

EXAMPLES:
---------
• "Weekly Status Template.docx" → Button: "Weekly Status Template"
• "Project Kickoff.docx" → Button: "Project Kickoff"
• "Monthly Review.docx" → Button: "Monthly Review"

INSTRUCTIONS:
-------------
1. Save your Word template files (.docx or .doc) in this folder
2. Restart the development server (npm run dev)
3. Templates will automatically appear as clickable buttons on the main page
4. Users can click a template to use it, or switch to "Upload Template" mode

NOTES:
------
• Only .docx and .doc files are recognized
• Files must be valid Word documents
• The application will analyze the template structure automatically
